Para criar um serviço web que retorna o IP e MAC address do usuário que está acessando, você precisará seguir estes passos no Rocky Linux 9:

*Passo 1: Instalar o servidor web*

Você pode escolher entre vários servidores web, como Apache, Nginx ou Lighttpd. Aqui, vamos usar o Apache.

```
sudo dnf install httpd
```

*Passo 2: Criar um script para obter IP e MAC address*

Crie um arquivo chamado `get_ip_mac.php` no diretório `/var/www/html/` com o seguinte conteúdo:

```
<?php
  $ip = $_SERVER['REMOTE_ADDR'];
  $mac = exec('arp -a ' . $ip . ' | awk \'{print $4}\'');
  echo "IP: $ip<br>MAC Address: $mac";
?>
```

*Passo 3: Configurar o Apache*

Habilite o servidor web e configure para executar o script PHP.

```
sudo systemctl enable httpd
sudo systemctl start httpd
```

*Passo 4: Testar o serviço*

Acesse o endereço IP do servidor no navegador.

```
http://<IP_DO_SERVIDOR>/
```

Você deve ver o IP e MAC address do seu dispositivo.

Observações:

- Certifique-se de que o firewall não esteja bloqueando o acesso ao servidor web.
- O comando `arp` pode não funcionar em todas as redes, especialmente se o servidor não tiver acesso ao mesmo segmento de rede que o cliente.
- Essa solução é simples e pode não ser segura para uso em produção.
